# Bet Book

A Chickadee Bandit app for long-term household bets, predictions, friendly wagers, receipts, and scorekeeping.

## Features

- Create long-running bets with multiple parties and predictions.
- Track wager terms, including bragging rights, money, chores, meals, custom prizes, and other agreements.
- Set review dates and see due-soon or overdue bets.
- Lock original predictions, then record later amendments separately.
- Attach receipts such as notes, links, screenshots, photos, or source-of-truth references.
- Mark a bet ready to judge, settle winners, void unresolved bets, and keep outcome notes.
- View a scoreboard with correct, incorrect, open, and voided counts for each participant.
- Filter the ledger by category, due soon, or overdue.

## Development

```bash
npm run dev
npm run test
npm run build
```

`npm run build` writes the installable bundle to `dist/bundle.json`.
